# 병원용 백오피스 개발 - (여신티켓)

<br/>

<img src="/img/about/portfolio/yeoshin/yeoshin_plus.png"/>

<br/>
<br/>

## 개요

**리부트** 프로젝트 중 일부로, 입점 병원이 사용하는 백오피스를 개발하였습니다.

<br/>

- 입점 병원용 백오피스 **(여신플러스)** 개발
- 예약 관리, 후기 관리, 정산, 광고 관리, 매출 등 다양한 페이지 개발 및 유지보수
- CPM 관련 포인트 결제 기능 추가
- 간단한 차트 기능 추가
